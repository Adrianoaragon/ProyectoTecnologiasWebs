Sobre el proyecto

Sistema de monitoreo de humedad de suelo en tiempo real, desarrollado como proyecto académico en la Universidad Simón Bolívar.

🎯

Objetivo

Recolectar datos de humedad de suelo mediante un sensor conectado a un ESP32, almacenarlos en una base de datos y visualizarlos en tiempo real a través de una plataforma web.

🔧

Tecnologías

El sistema está construido con Node.js en el backend, MySQL como base de datos, Plotly para visualización de datos y un ESP32 como microcontrolador del sensor.

📡

Hardware

Utilizamos un sensor de humedad de suelo capacitivo conectado a un ESP32, que envía lecturas periódicas al servidor mediante WiFi usando peticiones HTTP.

🔌

API Pública

Los datos recolectados estarán disponibles a través de una API pública para que otros estudiantes puedan acceder a la información del sensor sin necesidad de acceso directo a la base de datos.
