API Pública

Documentación de la API

Acceso libre a los datos del sensor de humedad de suelo. No requiere autenticación.

URL Base /api/publica

Endpoints disponibles

GET /api/publica/lecturas Últimas 50 lecturas

Retorna las últimas 50 lecturas registradas por el sensor, ordenadas de más reciente a más antigua.

Ejemplo de petición
fetch('/api/publica/lecturas')
  .then(r => r.json())
  .then(data => console.log(data))
Ejemplo de respuesta
{
  "ok": true,
  "total": 50,
  "datos": [
    {
      "id": 100,
      "humedad": 65.5,
      "sensor_id": "ESP32-01",
      "registrado_en": "2026-01-15T14:30:00.000Z"
    }
  ]
}

            

        


        
        

            

                GET
                /api/publica/ultima
                Última lectura
            

            

                
Retorna únicamente la lectura más reciente del sensor.

                

                    

                        
Ejemplo de petición

                        
fetch('/api/publica/ultima')
  .then(r => r.json())
  .then(data => console.log(data))

                    

                    

                        
Ejemplo de respuesta

                        
{
  "ok": true,
  "dato": {
    "id": 100,
    "humedad": 65.5,
    "sensor_id": "ESP32-01",
    "registrado_en": "2026-01-15T14:30:00.000Z"
  }
}

                    

                

                
                

            

        


        
        

            

                GET
                /api/publica/promedio
                Promedio del día
            

            

                
Retorna el promedio, máximo y mínimo de humedad del día actual.

                

                    

                        
Ejemplo de petición

                        
fetch('/api/publica/promedio')
  .then(r => r.json())
  .then(data => console.log(data))

                    

                    

                        
Ejemplo de respuesta

                        
{
  "ok": true,
  "dato": {
    "promedio": 58.3,
    "maximo": 75.0,
    "minimo": 42.1,
    "total_lecturas": 48,
    "fecha": "2026-01-15"
  }
}

                    

                

                
                

            

        


        
        

            

                GET
                /api/publica/historial?dias=7
                Historial por días
            

            

                
Retorna las lecturas de los últimos N días. Máximo 30 días.

                

                    dias
                    number
                    default: 7
                    Número de días a consultar (1-30)
                

                

                    

                        
Ejemplo de petición

                        
fetch('/api/publica/historial?dias=3')
  .then(r => r.json())
  .then(data => console.log(data))

                    

                    

                        
Ejemplo de respuesta

                        
{
  "ok": true,
  "dias": 3,
  "total": 144,
  "datos": [...]
}

                    

                

                
                

            

        


        
        

            

                GET
                /api/publica/stats
                Estadísticas generales
            

            

                
Retorna estadísticas globales de todos los datos históricos del sensor.

                

                    

                        
Ejemplo de petición

                        
fetch('/api/publica/stats')
  .then(r => r.json())
  .then(data => console.log(data))

                    

                    

                        
Ejemplo de respuesta

                        
{
  "ok": true,
  "dato": {
    "total_lecturas": 1500,
    "promedio_global": 61.2,
    "maximo_historico": 98.0,
    "minimo_historico": 5.0,
    "primera_lectura": "2026-01-01T08:00:00.000Z",
    "ultima_lectura": "2026-01-15T14:30:00.000Z"
  }
}

                    

                

                
                

            

        

    


    
    

        
Guía de uso rápido

        

            

                
JavaScript (Fetch)

                
const BASE = 'https://tu-url.ngrok-free.app';

async function obtenerHumedad() {
  const res  = await fetch(`${BASE}/api/publica/ultima`);
  const data = await res.json();
  console.log('Humedad:', data.dato.humedad + '%');
}

obtenerHumedad();

            

            

                
Python (requests)

                
import requests

BASE = 'https://tu-url.ngrok-free.app'

r = requests.get(f'{BASE}/api/publica/ultima')
data = r.json()
print('Humedad:', data['dato']['humedad'])
